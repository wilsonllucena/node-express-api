
# NODE-START

start de projetos em nodejs 



## Installation

Install project with npm

```bash
  git clone https://github.com/wilsonllucena/node-express-api
  cd node-express-api
  npm install 

```
    
## Running Tests

To run tests, run the following command

```bash
  npm run test:e2e
  npm run test:unit
```
